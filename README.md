# Galería de Fotos con Pexels

Esta Web App permite buscar, explorar y gestionar favoritos imágenes utilizando la API de Pexels. Fue desarrollada como proyecto.

## Funcionalidades Implementadas

### Página de inicio con categorías
Tres categorías principales (Naturaleza, Arte, Animales) que muestran imágenes al hacer clic, obtenidas directamente desde la API de Pexels.

### Buscador de imágenes
Búsqueda por palabras clave con validación mediante expresiones regulares para validar la palabra buscada.

### Filtro por orientación
Permite filtrar resultados por orientación: Todas, Verticales u Horizontales.

### Paginación
Navegación entre resultados para manejar grandes cantidades de imágenes.

### Favoritos
Añadir imágenes a favoritos con un botón.

Guardar favoritos en LocalStorage, manteniéndolos incluso después de cerrar o recargar la página.

Visualizar y eliminar favoritos.

### Responsive y compatible
El proyecto está diseñado con Mobile First y probada en Firefox, Chrome, Opera y Safari.

## Tecnologías Utilizadas

HTML5: Estructura semántica.

CSS3: Diseño responsivo y estilizado para distintos tamaños de pantalla.

JavaScript: Manejo de lógica, manejo de la API, almacenamiento local y validaciones.

Fetch API: Para hacer peticiones HTTP a Pexels.

LocalStorage: Para mostrar imágenes favoritas.
